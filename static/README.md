## Module : static

### Raison d’être

Ce module est responsable de la présentation visuelle de l’application web.
Il permet de définir l’apparence graphique de l’interface utilisateur sans modifier la logique de l'application.

### Fichiers principaux et responsabilités

- `style.css`  
  Définit les styles CSS utilisés par la page web, notamment la responsivité et l’apparence des boutons.

### Dépendances et hypothèses

**Hypothèses :**

- Les classes CSS définies correspondent aux éléments HTML du module templates.
